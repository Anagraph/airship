A Banner displays information that explains nearby content.

```react
<Banner color="red">
  <Banner.Icon>
    <Icon icon="alert_fill" color='#fff' size={12} />
  </Banner.Icon>
  <Banner.Content>
    <Text color='#fff'>
      An error has ocurred. Could not retreive customer information
    </Text>
  </Banner.Content>
</Banner>
```

### Props

#### **color** (string)

Change the background color.
