A Banner displays information that explains nearby content.

```react
<Banner color="red">
  <Banner.Icon>
    <AlertFillIcon color='#fff' width={12} height={12} />
  </Banner.Icon>
  <Banner.Content>
    <Text color='#fff'>
      An error has ocurred. Could not retreive customer information
    </Text>
  </Banner.Content>
</Banner>
```

### Props

#### **color** (string)

Change the background color.
