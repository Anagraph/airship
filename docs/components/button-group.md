Button group component explanation here.

```react
<ButtonGroup>
  <Button>
    <Icon icon="plus" />
  </Button>
  <Button>Hello</Button>
  <Button>
    World
  </Button>
</ButtonGroup>
```

### Props

#### **secondary** (boolean)

Theme the button group as a secondary one.

```react
<ButtonGroup secondary>
  <Button>
    <Icon icon="plus" />
  </Button>
  <Button>Hello</Button>
  <Button>
    World
  </Button>
</ButtonGroup>
```
